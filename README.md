# webt-log

#### Использование:
```javascript
log($text, $type);
```

#### Типы логов:
- Ошибка = 'error'
- Предупреждение = 'warning'
- Успех = 'success'
- Обычный вывод = 'default'

#### Примеры:
```javascript
log('Функция выполнена успешно!', 'success'); 
```
